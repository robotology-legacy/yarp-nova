Wrappers around YARP/ACE with a reduced, simpler interface.
For an operating systems class in UNIGE.

Requirements:

  ACE, YARP, ncurses [on linux]

  debian packages: libace-dev, libncurses-dev

Classes

  NovaSemaphore
    Simple semaphores

  NovaThread
    Simple threads

  NovaTime
    Time-related services

  NovaClient
    Socket client

  NovaTelnetClient
    Socket client specifically for talking to a server you
    could telnet too, like many MUDs
